그 밖의 글자를 다루는 태그들

일반글꼴 (태그로 감싸지 않는 텍스트)

b : 글자를 굵게 표시하는 태그

strong : 글자를 굵게 표시하는 태그 + 스크린리더
- 스크린리더 : 글자 데이터를 읽어줄때도 강조해서 읽어줌

i : 글자를 기울여서 표시하는 태크

em(emphasis) : 글자를 기울여서 표시하는 태그 + 스크린리더

mark : 형광펜 효과를 내는 태그

u : 글자애 밑줄이 그어지는 태그

s : 글자에 취소선을 그어주는 태그

small : 글자를 기본보다 작게 표현해주는 태그

기본글자에 sub : 아래첨자를 나타내는 태그
기본글자에 sup : 위첨자 를 나타내는 태그

abbr : 줄임말 표시
HTML란 월드 와이드 웹(World Wide Web)의 핵심 마크업 언어이다.

blockquote : 인용문
크게 성공한 사람은 그 성공에 비례할 만큼의 큰 노력이 숨어있다 - 더조은

글자 관련 태그 응용(태그를 중첩하여 사용 가능)

태그를 중첩으로 사용 가능하다.
b와 em을 사용하여 굵게, 기울임 표현 가능
s와 mark를 사용하여 취소선, 형관펜 표현 가능
